## Objective

Extract FingeringPatternControls and ScaleChordControls from App.tsx's duplicate mobile/desktop JSX blocks into reusable presentational components. Replace both occurrences of each block in App.tsx with the single extracted component, eliminating ~155 lines of duplicated JSX and reducing App.tsx by ~297 lines net. (Original estimate was ~138; actual reduction was larger because the research underestimated the full duplication scope.)

- @.vbw-planning/phases/01-extract-duplicated-controls/01-01-RESEARCH.md — complete props interfaces and JSX structures for both components (Component Extraction Details sections 1 and 2); shift-click multi-select logic detail for CAGED shapes; risk assessment
- @.vbw-planning/phases/01-extract-duplicated-controls/.context-lead.md — presentational-via-props decision; no state architecture changes

This plan depends on:
- Plan 01: `src/components/shared.css` must exist (components import it for .toggle-group, .control-section, etc.)
