# FretFlow v2.0 Plan C — Chord Input Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Use the **`frontend-design`** skill for UI-build tasks (Tasks 1, 3). Steps use checkbox (`- [ ]`) syntax.

**Goal:** In the Song-tab chord-edit pane, replace the separate Degree ToggleBar + Root NoteGrid pair with a single 12-note **degree grid** that shows the note name (prominent) and Roman numeral (small). Borrowed degrees render with muted numeral text. Auto-fill diatonic quality only on step creation; root and quality are independent after creation.

**Architecture:** A new `DegreeGrid` component renders 12 cells, one per chromatic note. Clicking an in-key note dispatches `updateProgressionStepDegreeAtom` (with `manualRoot: null`); clicking a borrowed (chromatic) note dispatches `updateProgressionStepRootAtom`. `addProgressionStepAtom` is updated to seed the new step's `qualityOverride` with the diatonic quality from `getDiatonicChord()` instead of `null`. `updateProgressionStepDegreeAtom` is changed to preserve the step's existing `qualityOverride` (v1 cleared it).

**Tech Stack:** React 19 + Jotai, CSS Modules, Tonal-backed `getDiatonicChord()` from `@fretflow/core`. Spec reference: [§6 of the design spec](../specs/2026-05-21-v2.0-redesign-design.md#6-subsystem-c--chord-input-redesign).

**Plan dependencies:** Plan C depends on Plan A's `groupProgression` heading and Song-tab Master-detail edit pane layout. Plan C is independent of Plan B.

```ts
import { getScaleNotes } from "./theory";

/**
 * Returns the set of in-key sharps-form note names for `scaleName` rooted at
 * `tonicNote`. The result is keyed by note name (no octave) and uses the
 * internal sharps-form convention (e.g. "A#" instead of "Bb").
 *
 * Used by the v2.0 DegreeGrid to decide whether a chromatic cell renders
 * as "in-key" (full prominence) or "borrowed" (muted numeral styling).
 */
export function getDiatonicNotes(
  scaleName: string,
  tonicNote: string,
): ReadonlySet<string> {
  const notes = getScaleNotes(scaleName, tonicNote);
  if (!notes || notes.length === 0) return new Set();
  return new Set(notes);
}
```

`getScaleNotes` already returns sharps-form notes via the Tonal wrapper at `packages/core/src/lib/tonal.ts`; no extra normalization is required.

- [ ] **Step 3: Update `addProgressionStepAtom`**

In `src/store/progressionAtoms.ts`, replace the body of `addProgressionStepAtom`:

```ts
import { getDegreeSequence, getDiatonicChord } from "@fretflow/core";
import { rootNoteAtom, scaleNameAtom } from "./scaleAtoms";

export const addProgressionStepAtom = atom(null, (get, set) => {
  const tonic = get(rootNoteAtom);
  const scaleName = get(scaleNameAtom);
  const previous = get(activeProgressionStepAtom);
  const sequence = getDegreeSequence(scaleName);

  // Pick the next ascending in-key degree after the previous step. Falls back
  // to the first in-key degree when there is no previous step (or it sits at
  // a borrowed root the sequence doesn't include).
  const previousIdx = previous ? sequence.indexOf(previous.degree) : -1;
  const nextIdx = previousIdx >= 0
    ? (previousIdx + 1) % sequence.length
    : 0;
  const degree = sequence[nextIdx] ?? "I";

  // Smart default quality: the diatonic chord on that degree.
  const diatonic = getDiatonicChord(degree, scaleName, tonic);
  const qualityOverride = diatonic?.quality ?? null;

  const next = [
    ...get(progressionStepsAtom),
    createProgressionStep({
      degree,
      duration: { value: 1, unit: "bar" },
      qualityOverride,
    }),
  ];
  set(progressionStepsAtom, next);
  set(activeProgressionStepIndexAtom, next.length - 1);
});
```

`src/components/shared/DegreeGrid.tsx`:

```tsx
import { useMemo } from "react";
import {
  NOTES,
  FLAT_KEYS,
  getDiatonicNotes,
  getDegreesForScale,
  getNoteDisplay,
} from "@fretflow/core";
import clsx from "clsx";
import styles from "./DegreeGrid.module.css";

export interface DegreeGridProps {
  scaleName: string;
  tonicNote: string;
  /** Currently selected note (sharps form). */
  selectedNote: string;
  /** Fires when an in-key cell is clicked. degree = "I", "ii", etc. */
  onSelectInKey: (note: string, degree: string) => void;
  /** Fires when a borrowed (out-of-key) cell is clicked. degreeLabel = "♭ii", etc. */
  onSelectBorrowed: (note: string, degreeLabel: string) => void;
  useFlats: boolean;
}

interface CellInfo {
  note: string;
  display: string;
  inKey: boolean;
  numeral: string;
}

// Borrowed-degree labels in major-key parlance. Indexed by semitone offset from
// the tonic. In-key offsets are filled by getDegreesForScale; borrowed offsets
// fall back to the table below.
const BORROWED_NUMERAL_BY_OFFSET: Record<number, string> = {
  1: "♭ii",
  3: "♭iii",
  6: "♯iv",
  8: "♭vi",
  10: "♭vii",
};

export function DegreeGrid({
  scaleName,
  tonicNote,
  selectedNote,
  onSelectInKey,
  onSelectBorrowed,
  useFlats,
}: DegreeGridProps) {
  const cells: CellInfo[] = useMemo(() => {
    const diatonic = getDiatonicNotes(scaleName, tonicNote);
    const degreesMap = getDegreesForScale(scaleName);
    const tonicIdx = NOTES.indexOf(tonicNote);

    return NOTES.map((note) => {
      const offset = (NOTES.indexOf(note) - tonicIdx + 12) % 12;
      const inKey = diatonic.has(note);
      const numeral = inKey
        ? degreesMap[offset] ?? ""
        : BORROWED_NUMERAL_BY_OFFSET[offset] ?? `${offset}`;
      return {
        note,
        display: getNoteDisplay(note, tonicNote),
        inKey,
        numeral,
      };
    });
  }, [scaleName, tonicNote]);

  return (
    <div className={styles.grid} role="group" aria-label="Chord root">
      {cells.map((cell) => (
        <button
          key={cell.note}
          type="button"
          className={clsx(
            styles.cell,
            cell.inKey ? styles.inKey : styles.borrowed,
            cell.note === selectedNote && styles.selected,
          )}
          data-in-key={cell.inKey ? "true" : "false"}
          aria-label={`${cell.display} ${cell.numeral}`}
          onClick={() =>
            cell.inKey
              ? onSelectInKey(cell.note, cell.numeral)
              : onSelectBorrowed(cell.note, cell.numeral)
          }
        >
          <span className={styles.note}>{cell.display}</span>
          <span className={styles.numeral} aria-hidden="true">
            {cell.numeral}
          </span>
        </button>
      ))}
    </div>
  );
}
```
