# FretFlow v2.0 Plan F Cleanup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Finish the v2.0 Inspector polish by (1) dropping hand-size logic, (2) tightening the close-voicing fret-span cap from 6 to 4, (3) re-exposing string-set selection as the canonical "where on the strings" picker, (4) surfacing the previously-invisible snap-to-scale-window toggle, (5) reverting an over-engineered connector primary/secondary dimming, (6) replacing rigid CSS-grid spans with auto-fit reflow, and (7) sizing short-value `LabeledSelect` triggers to their content.

**Architecture:** Surgical refactor across `packages/core/src/shapes/voicings.ts`, `src/store/chordOverlayAtoms.ts`, `src/components/ChordOverlayControls/*`, `src/components/Inspector/InspectorGrid.module.css`, and `src/components/LabeledSelect/*`. The note-highlight decoupling (already shipped in F5a/b+F5c/d) stays; the per-voicing selection model (also shipped) is rolled back in favor of per-string-set selection. The hand-size system is removed wholesale.

**Tech Stack:** React 19 + Jotai, CSS Modules, Vitest, Playwright. Repo conventions: `pnpm` for scripts, Conventional Commits with scope, `atomWithStorage` for persisted state, `data-*` attributes on PropGrid/LabeledSelect for CSS variants.

**Plan dependencies:** Builds on Plan F revision (committed 2026-05-22 by /Users/isaaccocar/.claude/plans/let-s-address-now-instead-buzzing-book.md). F1, F2, F4, F5a/b, F5c/d already shipped to branch `claude/elated-nobel-dd4e76` (PR #451). This plan covers F5w → F8.

describe("closeVoicings span cap", () => {
  it("never emits a candidate whose fretted-fret span exceeds 4", () => {
    const allChords = ["Major Triad", "Minor Triad", "Dominant 7th", "Major 7th"] as const;
    for (const chordType of allChords) {
      for (const chordRoot of NOTES) {
        const voicings = generateVoicings({
          chordRoot,
          chordType,
          tuning: STANDARD_TUNING,
          maxFret: 24,
          voicingType: "close",
        });
        for (const v of voicings) {
          const fretted = v.notes
            .map((n) => n.fretIndex)
            .filter((f) => f > 0);
          if (fretted.length < 2) continue;
          const span = Math.max(...fretted) - Math.min(...fretted);
          expect(span, `${chordRoot} ${chordType}: ${JSON.stringify(v.positionKeys)} span=${span}`).toBeLessThanOrEqual(4);
        }
      }
    }
  });

      chordType: "Dominant 7th",
      tuning: STANDARD_TUNING,
      maxFret: 24,
      voicingType: "close",

Create `src/store/voicingStringSets.ts`:

```ts
export const ALL_STRINGS: readonly number[] = [0, 1, 2, 3, 4, 5];

export interface StringSetOption {
  /** Stable identifier — "all" or "0-1-2-3" style. */
  id: string;
  /** String indices (0 = high E, 5 = low E in standard tuning). */
  strings: readonly number[];
}

export const ALL_STRINGS_OPTION: StringSetOption = {
  id: "all",
  strings: ALL_STRINGS,
};

/**
 * For a chord requiring `voiceCount` notes on a 6-string instrument,
 * enumerates every consecutive-string window plus an "all" sentinel.
 * Returns options in top-to-bottom string order (high strings first).
 */
export function buildStringSetOptions(voiceCount: number): StringSetOption[] {
  if (voiceCount < 3 || voiceCount > 5) return [ALL_STRINGS_OPTION];
  const sets: StringSetOption[] = [ALL_STRINGS_OPTION];
  const windowCount = 6 - voiceCount + 1;
  for (let start = 0; start < windowCount; start += 1) {
    const strings: number[] = [];
    for (let i = 0; i < voiceCount; i += 1) strings.push(start + i);
    sets.push({ id: strings.join("-"), strings });
  }
  return sets;
}
```

```tsx
import { useAtom, useAtomValue } from "jotai";
import { useMemo } from "react";
import {
  voicingStringSetAtom,
  stringSetOptionsAtom,
} from "../../store/chordOverlayAtoms";
import { useI18n } from "../../i18n/I18nProvider";
import { LabeledSelect } from "../LabeledSelect/LabeledSelect";

function labelForOption(id: string, t: ReturnType<typeof useI18n>["t"]) {
  if (id === "all") return t("inspector.chordStringSetAll");
  // "0-1-2-3" → "1·2·3·4" (guitar-numbered, dot separator)
  return id
    .split("-")
    .map((n) => String(parseInt(n, 10) + 1))
    .join("·");
}

export function ChordStringSetPicker() {
  const { t } = useI18n();
  const [value, setValue] = useAtom(voicingStringSetAtom);
  const options = useAtomValue(stringSetOptionsAtom);

  const items = useMemo(
    () =>
      options.map((o) => ({
        value: o.id,
        label: labelForOption(o.id, t),
      })),
    [options, t],
  );

  return (
    <LabeledSelect
      label={t("inspector.chordStringSetLabel")}
      hideLabel
      value={value}
      onChange={setValue}
      options={items}
      fit
    />
  );
}
```
