# Theming Pass Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Apply the user-provided light-mode reference palette to the fretboard note-color tokens, surface chrome, and lens-emphasis glows. Add two new tokens (`--note-glow-anticipation`, `--note-glow-hold`) that the post-Lens-Consolidation `getEmphasis` function references. Light-mode is the primary target; dark-mode is touched only for the two new glow tokens.

**Architecture:** Two structural changes (add two CSS variables, swap two hardcoded glow strings to CSS var references), then a token-value re-tune across six token families in `themes.css`'s `[data-theme="modern-light"]` block. Values come from the reference palette; `frontend-design` is invoked as the optional refinement step to iterate against the visual reference.

**Tech Stack:** CSS Custom Properties, TypeScript (semantics.ts edit), Vitest, Playwright (visual regression).

**Spec:** `docs/superpowers/specs/2026-05-27-theming-design.md`

**Sequencing:** Must land AFTER:
- Lens Consolidation plan (`docs/superpowers/plans/2026-05-27-lens-consolidation.md`) — provides the renamed `getEmphasis` function this plan edits.
- CAGED-E Recolor plan (`docs/superpowers/plans/2026-05-27-caged-e-recolor.md`) — provides the sky-blue E inherited state.

- [ ] **Step 2: Apply light-mode note-fill values**

Edit the light-mode block to use the values below (derived from the reference palette via `color-mix` and direct mapping):

```css
  /* Note fills — light mode, reference palette
     Tonic + chord-root in the warm ORANGE family;
     chord-tone in the cool CYAN family;
     scale-only in the soft PANEL-SOFT neutral;
     outside / tension in deeper ORANGE-family tints. */
  --fretboard-note-fill-tonic:          color-mix(in srgb, #b1431b 22%, #f6f2e9);
  --fretboard-note-fill-chord-root:     color-mix(in srgb, #b1431b 32%, #f6f2e9);
  --fretboard-note-fill-chord-tone:     color-mix(in srgb, #147088 18%, #f6f2e9);
  --fretboard-note-fill-scale-only:     #e3ddd8;
  --fretboard-note-fill-chord-outside:  color-mix(in srgb, #b1431b 40%, #f6f2e9);
  --fretboard-note-fill-tension-emph:   color-mix(in srgb, #b1431b 55%, #f6f2e9);
  --fretboard-note-fill-tension-strong: color-mix(in srgb, #b1431b 70%, #f6f2e9);
  --fretboard-note-fill-guide-emph:     color-mix(in srgb, #147088 35%, #f6f2e9);
```

The `color-mix(in srgb, X N%, base)` recipe tints the cream base with N% of the accent. Higher N = more saturated; lower N = more subtle. The N values above are an initial recommendation; frontend-design can iterate (Task 9).
